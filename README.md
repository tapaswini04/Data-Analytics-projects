# Excel-Bike-Sales-Analysis
This Excel project analyzes the Bike Buyers dataset to find factors that affect bike purchases. The data was cleaned and analyzed using pivot tables, and a dashboard with charts was created to show the insights. It demonstrates data analysis and visualization using Excel.

## Project Objective

The goal of this project is to analyze customer data to understand the factors that influence bike purchases.
This analysis helps identify patterns in **age, income, gender, and commute distance** that affect buying decisions.



## Questions (KPIs)

* Compare bike purchases based on **gender**.
* Identify which **age group** buys more bikes.
* Analyze **income level vs bike purchase**.
* Study the effect of **commute distance on bike buying behavior**.
* Compare purchases based on **marital status and region**.
* Identify patterns between **number of cars and bike purchase**.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Data Cleaning
* Interactive Dashboard
